# 项目管理工具

一款轻量级的多人项目管理网页工具，具备任务看板、实时更新、权限控制等核心能力，让团队成员可以自助更新状态，减少管理负担。

## 核心功能

### 1. 工作空间与项目
- 多项目/多空间支持：用户可加入或创建多个工作空间（团队），每个工作空间下包含多个项目。
- 项目模板：提供空白模板、敏捷开发模板、任务追踪模板等，快速初始化。
- 项目设置：成员管理、角色权限、项目阶段（待办、进行中、已完成）自定义。

### 2. 任务管理
- 任务看板：支持拖拽式看板（如“待处理→进行中→已完成”），可自定义列。
- 任务详情：标题、描述、指派人、截止日期、优先级（高/中/低）、标签、附件、子任务清单。
- 评论与动态：每条任务下可评论、@提及成员、查看操作历史。
- 批量操作：批量分配、批量移动、批量删除。
- 任务依赖：支持“阻塞/被阻塞”关系，防止进度冲突。

### 3. 用户与权限
- 统一身份认证：支持邮箱/密码、SSO（OAuth2.0）、企业微信/Lark 集成。
- 细粒度权限：工作空间所有者、项目管理员、成员、只读访客；对任务、文件、设置等分别授权。
- 团队组织：部门/小组层级管理。

### 4. 实时协作
- WebSocket 实时同步：多人同时查看/编辑任务，看板变化立即广播。
- 在线状态：显示成员当前在线状态，以及正在编辑的任务。
- 实时通知：浏览器推送（Web Push）或站内消息，任务指派、评论、截止提醒等。

### 5. 进度与统计
- 项目仪表盘：任务完成率、延期任务数、成员工作量热力图。
- 甘特图：展示任务时间线、依赖关系，支持拖拽调整。
- 燃尽图/累积流量图：适用于敏捷开发团队。
- 自定义报表：按项目、成员、标签、时间范围导出 CSV/PDF。

### 6. 文件管理
- 云存储集成：支持上传图片、文档、压缩包，支持预览（图片、PDF、Markdown）。
- 版本历史：记录文件每次修改，可回滚。
- 文件夹组织：按项目或任务归类文件。

### 7. 搜索与过滤
- 全局搜索：按任务名称、描述、评论内容、指派人、标签等检索。
- 高级过滤器：保存常用筛选条件（如“我的未完成任务”）。

### 8. 第三方集成
- Webhook / API：允许外部系统触发任务创建或同步数据。
- 即时通讯：与 Slack、钉钉、飞书机器人联动，发送任务通知。

## 功能亮点

1. **拖拽式任务看板**：直观的任务管理界面，支持拖拽调整任务状态。
2. **实时同步**：使用 WebSocket 实现多人实时协作，看板变化立即同步。
3. **数据可视化**：丰富的图表展示项目进度和团队工作量。
4. **细粒度权限**：灵活的角色权限系统，确保数据安全。
5. **响应式设计**：适配不同设备屏幕，提供良好的移动端体验。
